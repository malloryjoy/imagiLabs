The Concept

Imagi Labs is a fun, modern brand that invites future STEMMERS a chance to learn basic code concepts in a fun, playful environment. While the standard issue runs through a variety of programs from computer lab courses to in-classroom education, Imagi Labs has invented a stylish format to learn basic Python while creating a work of art!

Using the Imagi Labs Imagi Charm, along with its corresponding App, users can use Python matrices to create 8 by 8 pixelated animations that display on the screen of their Imagi Charm keychain. Using the App's social media capabilities, users can share their Designs and support their fellow creators as they go!

The Next Level

In getting familiarized with the app's functions, translating their real-life applications of Python in a somewhat re-invented way, I had a fun idea: what if I could create a Python application that generated a new charm image depending on the month? This app focused on getting down the basics, but it didn't involve much by way of the standard day-to-day functionalities of Python. Adding an if-elif-else function to the mix? Jackpot!

The Execution

From there, I created my imagery, using Imagi's IDE. Once published in the app, I saved the code for each one to convert to standard Python!

Now, how to display matrices as flashing animation? MatLabs and frames to the rescue! Once each design is converted, wrap them into individual/correspondingly named functions. The four leaf clover gets wrapped into a function titled "isMarch()". Beachy waves slip away like a moment in time within the function titled "isAugust()".

Now, wrap it all up with a bow... and by bow, I mean a simple if/else function. Declare and set a variable as the current month. If the variable matches the same numbered month of the year, run the function that displays the corresponding month's imagery. If January, light those fireworks. If it's June, let's see the birthday cake! Halloween? Jack O'Lanterns galore.
